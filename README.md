# POS - PC's Online Searcher

OCA大阪デジタルコンテンツ専門学校 1年次進級制作として開発した、校内PC稼働状況監視システムです。  
制作展にて実際の校内ネットワーク上で運用しました。

---

## 概要

MACアドレスをもとにネットワークスキャンを行い、教室ごとのPC使用状況をリアルタイムで可視化するWebアプリケーションです。

---

## 機能

- 教室ごとのPC稼働状況をリアルタイム表示
- MACアドレスベースのネットワークスキャン（Python）
- PC番号ごとのコメント編集機能
- 使用中・使用不可・空きの3状態表示
- レスポンシブデザイン対応

---

## システム構成

```
ブラウザ
    │
    ▼
PHP (Apache / XAMPP)
    │
    ├── MySQL（MACアドレス・教室情報を管理）
    │
    └── Python（ネットワークスキャン）
            │
            ▼
        校内ネットワーク上のPC（MACアドレスで稼働確認）
```

| 分野 | 技術 |
