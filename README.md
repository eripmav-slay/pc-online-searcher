# POS - PC's Online Searcher

1年次進級制作として開発した、校内PC稼働状況監視システムです。  
制作展示やその後実際の校内ネットワーク上で運用しました。
ネットワークに負荷をかけるやり方はあまりよろしくないのは理解していますが、当時作成した時のまま残しています。

---

## 概要

DBに登録したMACアドレスとARPテーブルで照合を行い、教室ごとのPC使用状況を可視化するWebアプリケーションです。

---

## 機能

- 教室ごとのPC稼働状況を表示
- MACアドレスベースのネットワークスキャン（Python）
- PC番号ごとのコメント編集機能
- 使用中・使用不可・空きの3状態表示

---

## システム構成

```
ブラウザ
    │
    ▼
PHP (Apache / XAMPP)
    │
    ├── MySQL（MACアドレス・教室情報を管理）
    │
    └── Python（ネットワークスキャン）
            │
            ▼
        校内ネットワーク上のPC（MACアドレスで稼働確認）
```
